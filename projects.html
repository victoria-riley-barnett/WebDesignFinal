My Flask blog

My other website, developed for a final exam for a class much like this one, is a Blog that uses Flask - a popular Python web application framework. Working with flask has been my preffered way of dipping my toes into web applications, because I am much more comfortable with Python than Javascript at the moment, so I currently prefer flask over something like Vue or React, though I'm sure I'd like to learn those eventually. I plan to continue to post to the blog in order to expand it's content, and also want to upgrade it's domain name and hosting solution soon - it is currently located at project-riley.herokuapp.com as a test deployment to ensure things are working correctly.

My Portfolio Site (this site!)
